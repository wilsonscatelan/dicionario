# Dicionário de Português

Projeto desenvolvido com o fim de aprimorar habilidades relacionadas a interface gráfica e listagem de componentes(estrutura de dados).

## Observações

O usuário dee possuir o arquivo de consulta "dic.text" na mesma pasta que o código principal(Dicionario.java). 

## Interface

O dicionário possui uma interface simples, formato quadrado com barra de busca.

## Funcionalidades

- Ao clilcar na barra de pesquisa "Buscar: ", o usuário pode pesquisar diversos tipos de palavras diferentes e ver seus respectivos significados.

- Ao digitar, antes de completar as palavras a barra de pesquisa informa possíveeis sugestões de palavras que o usuário pode estar buscando.

- O usuário pode navegar pelo campo de sugestões e procurar a palavra que deseja caso não queira digita-lá inteira ou busque palavras semelhantes.

## Tecnologias

- Java
- Bibliotecas swing e aws
